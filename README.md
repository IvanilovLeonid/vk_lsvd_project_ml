# Влияние контентных эмбеддингов на качество гибридных рекомендательных систем

Исследование того, как контентные эмбеддинги видео влияют на качество гибридных рекомендательных систем, на датасете VK-LSVD.

## Что сделано

- Реализованы три модели: **ItemKNN**, **ALS** (implicit) и **Hybrid BPR** (своя реализация на PyTorch)
- Контент подмешивался двумя способами: feature-augmentation внутри Hybrid BPR и ре-ранжирование топ-50 кандидатов ALS по content-similarity
- Протестированы размерности эмбеддинга 8, 32, 64
- Эксперименты прогнаны на 5 и 15 неделях обучающих данных, устойчивость проверена на 3 сидах

## Датасет

[VK-LSVD](https://huggingface.co/datasets/deepvk/VK-LSVD) — сабсемпл 10 000 пользователей, ~19 600 видео, данные по неделям.

## Основные результаты

- Лучший конфиг — Hybrid BPR с эмбеддингом dim=32: **+7.2% NDCG@10** к CF-only, **+44.5%** к ALS без контента
- Feature-augmentation (Hybrid BPR) работает лучше, чем ре-ранжирование поверх ALS
- Полезная размерность эмбеддинга зависит от объёма данных: мало данных → dim=8 (регуляризация), больше данных → dim=32 (лучше и стабильнее), dim=64 прироста не даёт
- Coverage вырос с ~29% (ALS) до ~50% (Hybrid BPR + контент)

## Ограничения

- Сабсемпл, не весь датасет — абсолютные значения метрик не переносятся на полный датасет
- Один тип контентного эмбеддинга (без разделения по модальностям)
- Только warm-start, cold-start отдельно не проверялся
- 3 сида — маловато для строгой статистики (p<0.05 не достигнут)
- Гиперпараметры не тюнились отдельно
